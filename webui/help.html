INI ⇄ Excel 转换工具帮助

本页提供当前 WebView 界面的基础使用说明，后续可继续补充更详细的转换规则文档。

基本流程

  1. 选择转换类型：INI → Excel 或 Excel → INI
  2. 通过界面按钮选择输入来源
  3. 选择输出目录，并填写输出文件名
  4. 点击“开始转换”执行任务

输入选择说明

  • 选择地图文件夹：适合直接选择已经拆解好的地图目录，程序会优先识别其中的 table 子目录。
  • 选择地图：用于后续直接选择 .w3x 地图文件，此功能依赖已正确配置 w3x2lni。

w3x2lni 配置说明

  • 设置时需要用户先选择 w3x2lni.exe
  • 程序会自动检查同目录下是否存在 w2l.exe
  • 真正写入配置文件的是 w2l.exe 的完整路径
