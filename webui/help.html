INI ⇄ Excel 转换工具帮助

此工具主要由AI完成，可能存在一些不完善之处，欢迎提issue反馈。

基本流程

  1. 选择转换类型：INI → Excel 或 Excel → INI
  2. 通过界面按钮选择输入来源
  3. 选择输出目录，并填写输出文件名
  4. 点击“开始转换”执行任务

输入选择说明

  • 选择地图文件夹：选择你LNI格式的地图目录，程序会优先识别其中的 table 子目录。
  • 选择地图：直接选择 .w3x 地图文件，此功能依赖你正确配置 w3x2lni（该功能未完成）。
